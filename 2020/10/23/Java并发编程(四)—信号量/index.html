Java并发编程基础(四)——信号量

Posted on 2020-10-23 Edited on 2020-10-24 In Java Views: Valine:
《Java并发编程艺术mini》and 极客《Java并发编程实战》，信号量实现对象池

信号量和互斥锁最大的差别在于信号量允许多个线程进入临界区，这一特性可以用在一些池化机制上面。

信号量一般包括：

  • 计数器
  • init方法，初始化计时器
  • up方法，计数器加一
  • down方法，计数器减一
  • 等待队列。等待信号量的线程队列
